Departures Board Setup
You will need a National Rail Enquiries Darwin Lite Webservice token if you want to display National Rail departures. Enter the token exactly as received (in the format nnnnnnnn-nnnn-nnnn-nnnn-nnnnnnnnnnnn). You can register for a token free of charge here.
Enter your openweathermap.org api key if you want to display station weather information. You can register for a free api key here.
Optionally, enter your Transport for London api key (app_key) if you have one (the board will operate fine without one). You can register for a free api key here.
